# 🔐 Security Policy  
### Keeping Stays and its users safe

The Stays project takes security seriously. We are committed to protecting user data, maintaining platform integrity, and responding quickly to vulnerabilities.

---

## 🛡️ Supported Versions

Security updates apply to:

- `main` (stable branch)  
- The most recent tagged release  
- Any actively deployed environment  

Older branches may not receive security fixes unless explicitly maintained.

---

## 🐞 Reporting a Vulnerability

If you discover a security issue, **please report it responsibly**.

### **Do NOT open a public GitHub issue.**

Instead, contact the maintainers privately:

- Email: **[security@stays.app]** (placeholder — replace with real address)
- Or message a maintainer directly if you have an established channel

Please include:

- A clear description of the vulnerability  
- Steps to reproduce  
- Potential impact  
- Any suggested fixes (optional but appreciated)

We aim to acknowledge reports within **72 hours**.

---

## 🔒 Responsible Disclosure Expectations

We ask that you:

- Give us reasonable time to investigate and fix the issue  
- Avoid exploiting the vulnerability  
- Avoid accessing or modifying user data  
- Avoid publicly disclosing details until a fix is released  

We will:

- Investigate promptly  
- Keep you updated  
- Credit you (if desired) in release notes  
- Publish a transparent summary after resolution  

---

## 🧰 Security Best Practices for Contributors

When contributing code:

- Validate all inputs  
- Avoid storing sensitive data in logs  
- Use parameterized queries  
- Follow OWASP guidelines  
- Never commit secrets or credentials  
- Use HTTPS for all network communication  
- Follow platform‑specific security patterns (e.g., Keychain, Keystore)  

---

## 🧪 Security Testing

We encourage contributors to:

- Run static analysis tools  
- Add tests for authentication/authorization flows  
- Review API endpoints for least‑privilege access  
- Validate file uploads and media handling  

---

## ❤️ Thank You

Security is a shared responsibility.  
We appreciate your help in keeping Stays safe for everyone.
